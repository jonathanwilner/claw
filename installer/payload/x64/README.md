# x64 Payload Slot

This directory is reserved for x64-specific bundle content.

Examples of future staged assets:

- native Windows helpers compiled for x64
- x64-only installer payloads
- architecture-specific configuration overrides
Place x64-specific offline payloads here.

Supported conventions:

- `wsl.msi`
- `DockerDesktopInstaller.exe`
- `images/*.tar`
- `ollama-models/ollama-models.tar.gz`
